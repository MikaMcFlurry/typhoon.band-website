# Phase 04 – Admin Content CRUD

## Goal

Allow admins to edit text/content records that already exist in the data model.

## Build CRUD for

- site settings
- hero content
- band info
- members text/order/visibility
- shows
- platform links
- SEO entries
- legal pages

## Requirements

- publish/visibility toggles
- sort order where applicable
- locale support de/en/tr
- safe validation
- no broken public fallback
- preview current values

## Roles

Owner:
- everything

Admin:
- content, shows, platform links, SEO

Legal pages:
- preferably owner-only or clearly restricted

## Do not build

- complex media/audio uploads
- shop
- payment
