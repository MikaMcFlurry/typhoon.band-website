# Phase 03 – Admin Auth + Dashboard

## Goal

Create the secure Admin foundation.

## Build

- Supabase Auth login
- protected admin layout
- session handling
- active admin profile check
- owner/admin role helpers
- logout
- dashboard overview
- no public access to admin pages

## Dashboard modules

Show placeholders/cards for:
- Booking
- Shows
- Music
- Gallery
- Members
- Legal
- SEO
- Platform links
- Settings

## Security

- route protection server-side
- no client-only auth protection
- no service role in client
- inactive admin denied
